<h2><code>DROP TABLE</code></h2><p>Quando uma tabela não é mais útil e não precisamos dela, é conveniente excluí-la; desta forma não ocupará espaço e poderemos usar seu nome em outra ocasião.</p><p>Para eliminar uma tabela usaremos o comando <a href="https://www.postgresql .org/docs/14/sql-droptable.html" target="_blank"><code>DROP TABLE</code></a>.</p><h3>Sintaxe:</h3><blockquote>< pre><code>DROP TABLE TableName [ CASCADE];</code></pre></blockquote><p>Esta instrução excluirá a tabela do banco de dados incluindo seus dados (linhas). Todas as informações existentes dessa tabela no Dicionário de Dados também serão excluídas.</p><h3>Exemplo:</h3><p>Se criamos uma tabela <code>USERS</code> com o comando: < blockquote><pre><code>CREATE TABLE USERS (<br /> Login VARCHAR (25)<br /> CONSTRAINT Users_log_PK PRIMARY KEY<br />);</code></pre></blockquote><p > Poderíamos removê-lo com o seguinte comando:</p><blockquote><pre><code>DROP TABLE USERS;</code></pre></blockquote><p>Tenha cuidado ao usar este comando, excluindo um A tabela é irreversível e não há solicitação de confirmação antes de ser executada.</p><h3>Exercício:</h3><p>Em um dos exercícios de criação de tabelas, solicitamos a criação dos Módulos da tabela:</p ><blockquote><pre><code>CREATE TABLE MODULES (<br /> name VARCHAR(100) PRIMARY KEY,<br /> hours INTEGER<br />);</code></pre></ blockquote> <p>Como você excluiria a tabela <code>MODULES</code>?</p>
